Webcomic Controls
=================
Navigate through webcomic pages using your arrow keys!

The button icon displays enabed (green) and disabled (red) status.


Controls
========
left+right: enable/disable
left: back/previous
right: next
ctrl+up: random
ctrl+left: first
ctrl+right: last
